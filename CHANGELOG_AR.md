# سجل التحديثات

## الإصدار 0.3.0

- إضافة بحث فوري في عنوان التكليف والمادة والدكتور والتعليمات والمهام.
- إضافة صفحة مستقلة للتكليفات مع فلاتر للحالة والأولوية والنوع.
- إضافة ترتيب حسب أقرب موعد أو أحدث إضافة أو مستوى الإنجاز.
- تفعيل صفحة المواعيد وعرض تكليفات اليوم والأسبوع والمتأخرة.
- إضافة إدخال اسم الدكتور أو المدرب.
- السماح بإضافة خطوات عمل مخصصة عند إنشاء التكليف.
- تحسين بطاقات التكليفات وإظهار النوع والحالة والتأخير.
- إضافة استعادة كلمة المرور من شاشة الدخول.

## الإصدار 0.2.0

- إضافة التسجيل وإنشاء الحساب واستعادة كلمة المرور عبر Supabase.
- إضافة المزامنة السحابية للتكليفات والمهام.
- إضافة تعديل الاسم وتسجيل الخروج والمزامنة اليدوية.
- إضافة حذف التكليفات والمهام.
- تصحيح المعرّفات لتكون UUID متوافقة مع PostgreSQL.
- تحسين مخطط قاعدة البيانات وسياسات RLS.
- إنشاء Storage bucket وسياسات وصول خاصة بالمستخدم.
- إضافة مؤشرات الاتصال والمزامنة ورسائل الأخطاء العربية.
- الإبقاء على الوضع المحلي عند عدم إعداد Supabase.
