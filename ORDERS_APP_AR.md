# مدار الطلبات

تطبيق عربي مستقل في الاستخدام داخل مستودع منجِز، مخصص لإدارة طلبات العملاء والمهام والمدفوعات من الهاتف.

## الرابط المتوقع

`https://madardigital.github.io/munjiz/orders/`

## الخصائص في الإصدار 1.0

- لوحة تحكم للطلبات النشطة والمتأخرة ومهام اليوم والمبالغ المتبقية.
- إضافة الطلب مع العميل ورقم واتساب والخدمة والموعد والسعر والمدفوع.
- خطوات تنفيذ تلقائية حسب نوع الخدمة.
- البحث والتصفية حسب الحالة والخدمة.
- إدارة المهام الشخصية والمرتبطة بالطلبات.
- تسجيل الدفعات وعرض المتبقي وسجل المدفوعات.
- سبعة قوالب رسائل واتساب قابلة للتعديل والنسخ.
- تخزين سحابي وسياسات RLS لكل مستخدم عبر Supabase.

## البنية التقنية

- الواجهة: React + TypeScript + Vite.
- قاعدة البيانات والمصادقة: Supabase.
- النشر: GitHub Pages ضمن صفحة متعددة المداخل.
